react-mvi

Minimal framework for react + rxjs mvi architecture

View on GitHub Download .zip Download .tar.gz

Simple and useful framework for reactjs and rxjs.

Construct rxjs stream flow around React Component. These construction is able to without this framework, but it's easily become so complex and so dirty. This framework solve minor problems that reactjs and that ecosystems faced.

1 Complex Props and State.

react-mvi solve complex props managements. All props are constructed by service and, all components have not state. So all React Component is STATELESS.

2 Complex event handling.

react-mvi solve complex event managements by dependency injection and reactjs context system. All event are fired by IO modules that exists in Context. That IO modules was injected to context by DI container.

3 RxJS combination.

react-mvi has component named Subscriber. That component extract RxJS Observable exists in component props or children. So you do not need thinking about props include Obsevables.
